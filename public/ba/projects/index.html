Ispod su neki od mojih projekata na kojima sam radio prethodnih godina, a koji su objavljeni na internetu.

vactija

GitHub

Vaktija za command-line interface na Linux i macOS sistemima implementirana u C-u. Koristi podatke ponuđene na vaktija.ba, stoga je korisna samo unutar Bosne i Hercegovine. Lako se integriše u skripte.

Lavinija

GitHub

Sada već napušteni projekt prevođenja Whitakerog WORDS - rječnika i alata za leksičku analizu latinskog jezika - u moderniju Kotlin verziju. Ima najosnovniju funkcionalnost, no izgubio sam interes da dalje radim na njemu.

whitaker-words-jsonisator

GitHub

Python skripta koja je prvobitno trebala biti korištena za Laviniju. Uzima sve WORDS podatke i pretvara ih u odgovarajući JSON format, što omogućava lakšu integraciju sa modernim softverom.

csvn

GitHub

Rudimentarni CSV parser napisan koristeći ANSI C inspirisan pristupom koji jsmn koristi za JSON. Podržava RFC 4180 CSV sa dodatnim opcijama.

PMFStableMarriage

GitHub

Python implementacija Gale-Shapley algoritma. Prvobitno namijenjena za mene i kolege na predmetu Matematičke osnove kompjuterske nauke na PMF Tuzla kako bismo lakše vizualizirali sam algoritam i analizirali ga, upotrebljiva je i za praktično svaki projekt koji je treba.

Ouroboros

GitHub

Spigot plugin napisan u Java jeziku za jedan privatni Minecraft server. Uvodi mehaniku života za igrače inspirisana klasičnim igrama kao što je Super Mario. Podržava YAML i SQL za pohranu podataka. Trenutno se ne održava.

pixel-warzone

GitHub

Pixelmon/Sponge plugin napisan u jeziku Java za jedan privatni Minecraft server. Implementira free-for-all zonu unutar koje igrači mogu izgubiti svoje Pixelmone ukoliko izgube bitku.
